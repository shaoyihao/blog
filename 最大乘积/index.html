原题链接：最大乘积

题目大意：将正整数 nnn（n≥2n\ge 2n≥2）分解成若干个互不相同的自然数的和，且使这些自然数的乘积最大。

背景知识：整数划分

设 SSS 为最佳拆分方案，其中最小元为aaa，最大元为bbb，则有以下几个事实：

  • a>1a>1a>1。

    若a=1a=1a=1，则把111和bbb去掉，替换为b+1b+1b+1，总和不变，而乘积变大，和“SSS是最佳方案”相矛盾。

  • a<5a< 5a<5。

    若a≥5a\ge 5a≥5，则aaa可以继续拆分【a=2+(a−2)a=2+(a-2)a=2+(a−2)】，乘积可以更大，矛盾。

    若(a,b)(a,b)(a,b) 范围内存在多个正整数不在SSS中，假设∃x,y∈(a,b)\exist x,y\in(a,b)∃x,y∈(a,b)且均∉S\notin S∈/​S。不妨设x<yx<yx<y。设 ppp 为 SSS 中最大的<x<x<x的正整数（可得 p+1∉Sp+1\notin Sp+1∈/​S），qqq 为 SSS 中最小的>y>y>y的正整数（可得 q−1∉Sq-1\notin Sq−1∈/​S），有：q>y>x>pq>y>x>pq>y>x>p，因此 q−p≥3q-p\ge 3q−p≥3，则：

    (p+1)(q−1)=pq+q−p−1≥pq+2>pq(p+1)(q-1)=pq+q-p-1\ge pq+2>pq (p+1)(q−1)=pq+q−p−1≥pq+2>pq

    因此将 p,qp,qp,q 换成 p+1,q−1p+1,q-1p+1,q−1 可以使乘积更大，矛盾。

  • a=4 ⟺ a=b=n=4a=4\ \Longleftrightarrow\ a=b=n=4a=4 ⟺ a=b=n=4。

    a=4a=4a=4 时，则一定有 5∉S5\notin S5∈/​S，因为将 555 换成 2+32+32+3，可以使乘积更大，矛盾。也一定有 6∉S6\notin S6∈/​S，因为将 4+64+64+6 换成 2+3+52+3+52+3+5 可以使总乘积增大，矛盾。而若 b≥7b\ge 7b≥7，则根据第三点可得 5,65,65,6至少有一个在SSS中，产生矛盾。

    因此 b=4b=4b=4。

对于 nnn（n≥5n\ge 5n≥5） ，其最佳拆分方案一定是上述4种形式之一。因为上述集合 SSS 和 元素和nnn 之间是一种 一一映射，因此当 nnn 确定时，其对应的集合形式 SSS 是唯一的。所以我们只需知道 nnn 是由哪种算式计算而来，便可得出其最佳拆分方案。

#include <iostream>
#include <vector>
using namespace std;

const int N = 10010;
bool st[N]; // st[i]=true表示最佳划分方案中含有该和数

int main()
{
    int n;
    cin >> n;

    int t = 2, sum = 0;
    while (1) 
    {
        st[t] = true;
        sum += t;
        if (sum + t + 1 > n) break;
        t ++;
    } //此时 sum = 2+...+t ≤ n < 2+...+t+t+1
    
    //微调最佳方案
    int k = n - sum;
    if (k == t) st[2] = false, st[t + 2] = true;
    else if (k == t - 1) st[2] = false, st[t + 1] = true;
    else if (1 <= k && k <= t - 2) st[t - k + 1] = false, st[t + 1] = true;

    //求最佳方案的乘积（高精度）
    vector<int> res;
    bool first = true;
    for (int i = 1; i <= n; i ++)
    {
        // if (st[i] == true) cout << i << ' '; 输出最佳方案
        if (st[i] == true && first) 
        {
            res = vector<int>(1, i);
            first = false;
        }
        else if (st[i] == true) res = mul(res, i);
    }
    //puts("");
    //for (int i = res.size() - 1; i >= 0; i --) cout << res[i];
    return 0;
}
