2的幂次方表示

· 2022-08-30 · # 递归

原题链接：2的幂次方表示

题目大意：给定一个正整数，输出该数的2的幂次方表示。例如：

137=27+23+20=2(7)+2(3)+2(0)=2(22+2+20)+2(2+20)+2(0)=2(2(2)+2+2(0))+2(2+2(0))+2(0)137=2^7+2^3+2^0\\=2(7)+2(3)+2(0)\\=2(2^2+2+2^0)+2(2+2^0)+2(0)\\=2(2(2)+2+2(0))+2(2+2(0))+2(0) 137=27+23+20=2(7)+2(3)+2(0)=2(22+2+20)+2(2+20)+2(0)=2(2(2)+2+2(0))+2(2+2(0))+2(0)

思路：对于一个数 XXX，如何快速地将其转化为 2a+2b+2c+...2^a+2^b+2^c+...2a+2b+2c+... 的形式？我们很自然地就会想到二进制，例如13=1101(2)13=1101_{(2)}13=1101(2)​在第0,2,30,2,30,2,3位上是1，因此 13=23+22+2013=2^3+2^2+2^013=23+22+20 。

可以借助STL中的 bitset 来实现判断 XXX 的二进制形式中某一位是否为 111 ：

bitset<16> a(X); 
for (int i = 15; i >= 0; i --)
    if (a.test(i)) // 判断第i位是否为1
    {
        ...
    }

此处函数dfs(u)的功能就是求出参数uuu的 2的幂次方表示。首先将 uuu 转化为 2a+2b+2c+...2^a+2^b+2^c+...2a+2b+2c+... 的形式，再判断是否需要对a,b,c,...a,b,c,...a,b,c,...继续分解：注意到最终的2的幂次方表示中，单位项只有：2,2(0)2,2(0)2,2(0) 这2种，因此 iii 只要大于111，那它就需要再细分。【只有满足一定情况，才进行递归处理】

由于一个正整数最终一定会拆分至 2、12、12、1，因此最终递归一定会结束。

#include <iostream>
#include <string>
#include <bitset>
using namespace std;

string res;

void dfs(int u)
{
	bitset<16> a(u); //将u转换为bitset处理
	
    bool isFirstOne = true;
	for (int i = 15; i >= 0; i --) //从高位到低位
		if (a.test(i)) 
		{
			if (isFirstOne == false) res += "+"; //若不是首项，则前面要添一个'+'
			else isFirstOne = false;
			
            if (i == 0) res += "2(0)";
			else if (i == 1) res += "2";
			else // i > 1，可以继续分解，递归下去
			{
				res += "2(";
				dfs(i);
				res += ")";
			}
		}
}

int main()
{
	int x;
	cin >> x;
    dfs(x);
    cout << res;
	return 0;
}
