砍树

· 2022-09-08 · # 二分

原题链接：砍树

题目大意：

思路：二分查找HHH的边界，使：

max⁡{x1−H, 0}+max⁡{x2−H, 0}+...+max⁡{xn−H, 0}≥M\max\{x_1-H, \ 0\}+\max\{x_2-H, \ 0\}+...+\max\{x_n-H, \ 0\}\ge M max{x1​−H, 0}+max{x2​−H, 0}+...+max{xn​−H, 0}≥M

当 HHH 过大时，总长度为小于 MMM，当 HHH 较小时，总长度又超出 MMM，因此要查找的就是左区间的右端点：

#include <iostream>
using namespace std;

const int N = 1000010;
int q[N], n, m;

bool check(int x) //判断高度设置为x时能否得到总长度为m的木材
{
    long long res = 0;
    for (int i = 0; i < n; i ++)
        res += max(q[i] - x, 0);
    
    if (res >= m) return true;
    else return false;
}

int main()
{
    cin >> n >> m;
    for (int i = 0; i < n; i ++) cin >> q[i];

    int l = 0, r = 1e9;
    while (l < r)
    {
        int mid = (l + r + 1) / 2; 
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    cout << l;
    
    return 0;
}
