两（三）数之和

· 2022-09-13 · # 双指针

原题链接：两数之和

题目大意：给定一个含nnn个元素的序列，找出其中一组和为ttt的两个元素。（保证一定有解）

思路：首先将该数列进行递增排序，此时我们来研究一下该数列中蕴含的单调性：令i,ji,ji,j（i<ji<ji<j）分别指向数列中的2个元素，则

  • 若 q[i]+q[j]>tq[i]+q[j]>tq[i]+q[j]>t，则 q[i]+q[j→]q[i]+q[j\rightarrow]q[i]+q[j→] 也一定大于ttt
  • 若 q[i]+q[j]<tq[i]+q[j]<tq[i]+q[j]<t，则 q[←i]+q[j]q[\leftarrow i]+q[j]q[←i]+q[j] 也一定小于ttt

算法步骤为：

  • 初始时让 iii 指向首元素，jjj 指向尾元素。
  • 若q[i]+q[j]>tq[i]+q[j]>tq[i]+q[j]>t，则j --
  • 若q[i]+q[j]<tq[i]+q[j]<tq[i]+q[j]<t，则i ++

可以证明：对于排好序且有解的数组，该双指针法一定能遍历到最优解。

设最优解的下标分别为 l, rl,\ rl, r。由于初始时 iii 指向首元素，jjj 指向末元素，则 i≤l<r≤ji\le l<r\le ji≤l<r≤j，i,ji,ji,j指针在l,rl,rl,r两侧。根据上述算法步骤，i,ji,ji,j 指针会逐渐向中间靠拢：

  • 若右指针已经移动到了 rrr 时，左指针仍在 lll 左边，则此时q[i]+q[j]<tq[i]+q[j]<tq[i]+q[j]<t，

    算法会一直执行i ++直到 i=li=li=l，达到最优解。

  • 若左指针已经移动到了 lll 时，右指针仍在 rrr 右边，则此时q[i]+q[j]>tq[i]+q[j]>tq[i]+q[j]>t，

    算法会一直执行j --直到 j=rj=rj=r，达到最优解。

每轮迭代指针必须移动一个，所以最终双指针一定会收敛在 l,rl,rl,r。

【注意到：双指针在任何时候都不可能处于(l,r)( l , r )(l,r) 之间】

时间复杂度：O(n)O(n)O(n)

#include <iostream>
#include <algorithm>
using namespace std;

const int N = 100010;
int q[N];

int main()
{
    int n, t;
    cin >> t >> n;
    for (int i = 1; i <= n; i ++) cin >> q[i];
    sort(q + 1, q + 1 + n);

    int i = 1, j = n;
    while (i < j) //循环最多进行到将数组完整扫描一遍（两指针碰撞）
    {
        if (q[i] + q[j] > t) j --;
        else if (q[i] + q[j] < t) i ++;
        else break;
    }

    cout << q[i] << ' ' << q[j];
    return 0;
}

另一种写法：

for (int i = 1, j = n; i <= n; i ++)
{
    while (j > i && q[i] + q[j] > target) j --;
    if (q[i] + q[j] == target)
    {
        cout << q[i] << ' ' << q[j];
        break;
    } 
}

若该数列具有多组解，则解的分布一定满足：l1<l2<⋯<lk<rk<⋯<r2<r1l_1<l_2<\cdots<l_k<r_k<\cdots<r_2<r_1l1​<l2​<⋯<lk​<rk​<⋯<r2​<r1​，呈现一种层层嵌套的特点。本算法所找出的最优解就是最外层的那个解。

若要找出所有解，只需再找出一个解后，再将指针移动一下即可（打破平衡）。

int i = 1, j = n;
while (i < j)
{
    if (q[i] + q[j] > t) j --;
    else if (q[i] + q[j] < t) i ++;
    else 
    {
        cout << q[i] << ' ' << q[j] << endl;
        i ++; //i++或j--任写一个（只要让指针向中间移动一下即可）
    }
}

原题链接：三数之和

题目大意：给定一个含nnn个元素的序列，找出所有 和为ttt的三个元素。（保证一定有解）

先固定一个元素q[i]q[i]q[i]，再利用双指针法在剩余元素中寻找2个元素满足元素和为t−q[i]t-q[i]t−q[i]。

时间复杂度为：O(n2)O(n^2)O(n2)

#include <iostream>
#include <algorithm>
using namespace std;

const int N = 100010;
int q[N];

int main()
{
    int n, t;
    cin >> t >> n;
    for (int i = 1; i <= n; i ++) cin >> q[i];
    sort(q + 1, q + 1 + n);

    for (int i = 1; i <= n; i ++) //“固定”i
    {
        int l = i + 1, r = n;
        while (l < r)
        {
            if (q[l] + q[r] == t - q[i]) 
            {
                cout << q[i] << ' ' << q[l] << ' ' << q[r] << endl;
                r --;
            }
            else if (q[l] + q[r] > t - q[i]) r --;
            else l ++;
        }
    }
    return 0;
}
