机器人跳跃

· 2022-09-08 · # 二分

原题链接：机器人跳跃

题目大意：n+1n+1n+1座塔的高度存储于hhh数组，其中h[0]=0h[0]=0h[0]=0（起点）。
机器人起始时位于第000座塔，初始能量为E0E_0E0​，接下来会一座一座地在塔间跳跃。机器人在第iii座塔上时的能量为E[i]E[i]E[i]，满足：

E[i+1]={E[i]+∣E[i]−h[i+1]∣,E[i]≥h[i+1]E[i]−∣E[i]−h[i+1]∣,E[i]<h[i+1]=2E[i]−h[i+1]E[i+1]=\left\{\begin{matrix} E[i]+|E[i]-h[i+1]|, & E[i]\ge h[i+1]\\ E[i]-|E[i]-h[i+1]|,&E[i]<h[i+1]\end{matrix}\right.=2E[i]-h[i+1] E[i+1]={E[i]+∣E[i]−h[i+1]∣,E[i]−∣E[i]−h[i+1]∣,​E[i]≥h[i+1]E[i]<h[i+1]​=2E[i]−h[i+1]

若存在某座塔ttt（t≤nt\le nt≤n），有：E[t−1]<h[t]−E[t−1]E[t-1]<h[t]-E[t-1]E[t−1]<h[t]−E[t−1]，即：E[t]<0E[t]<0E[t]<0，则机器人将无法到达终点（第nnn座塔）。

求要想使机器人能够到达终点，初始能量最少应为多少？

思路：当E0E_0E0​很大时，显然可以到达终点；当E0E_0E0​较小时就会无法到达终点。容易看出其中的单调性，答案是右区间的左端点。

基本框架：

int l = 0, r = 1e5; //答案的范围
while (l < r)
{
    int mid = (l + r) / 2;
    if (check(mid)) r = mid;
    else l = mid + 1;
}
printf("%d", r);

关键是check函数如何写？即如何判断给定的一个初始能量能否使机器人到达终点。
方法：利用给定的E0E_0E0​，根据题意，递推求出在各塔上时的能量E[i]E[i]E[i]，若存在E[i]<0E[i]<0E[i]<0，则说明不能到达终点；否则说明可以。

注意到：当各塔之间高度相差不大时，近似有：E[i]=2E[i−1]E[i]=2E[i-1]E[i]=2E[i−1]，此时EEE值以指数递增，很容易爆整数类型。

事实上当存在某塔处 E[t]≥max⁡1≤i≤n{hi}E[t]\ge\max\limits_{1\le i\le n}\{h_i\}E[t]≥1≤i≤nmax​{hi​}，之后E[t+1]=2E[t]−h[t+1]≥E[t]E[t+1]=2E[t]-h[t+1]\ge E[t]E[t+1]=2E[t]−h[t+1]≥E[t]，即之后EEE值总是不减，始终大于0，即之后不可能存在E<0E<0E<0的情况，一定可以到达终点。

bool check(int e)
{
    for (int i = 1; i <= n; i ++) //递推求E值
    {
        e = e * 2 - h[i];
        if (e < 0) return false;
        if (e >= 1e5) return true; //一定可以到达终点
    }
    return true;
}
